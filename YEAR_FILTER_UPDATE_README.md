### מה השתנה?

**לפני:**
- דוחות הציגו **את כל המשפחתונים** בהיסטוריה

**אחרי:**
- דוחות מציגים רק משפחתונים מ-**365 ימים אחורה**

## ?? אילו דוחות מושפעים?

### ? דוחות עם סינון שנה אחורה:

1. **משפחתונים שנסגרו מיידית** (`approvalStatus = 2`)
2. **משפחתונים שצריכים תיקון ליקויים** (`approvalStatus = 0`)
3. **משפחתונים שקיבלו אישור מיידי** (`approvalStatus = 1`)

### ? דוחות ללא סינון זמן:

4. **משפחתונים שלא נבדקו עדיין** - אין להם תאריך מבדק
5. **משפחתונים ששינו כתובת** - רלוונטי לכל ההיסטוריה
6. **באיזה משפחתונים ביקר יועץ** - סינון לפי תאריך ספציפי

## ?? בדיקה

### איך לבדוק?

1. **כנסי לדוחות** ? "משפחתונים שנסגרו מיידית"
2. **בדקי** שמופיעה ההודעה: "?? הדוח מציג משפחתונים מהשנה האחרונה"
3. **ודאי** שהטבלה מציגה רק מבדקים מהשנה האחרונה

### דוגמת בדיקה ב-SQL:

```sql
-- בדיקה ידנית - האם כל התאריכים מהשנה האחרונה?
SELECT 
    kindergartenCode,
    auditDate,
    DATEDIFF(DAY, auditDate, GETDATE()) AS DaysAgo
FROM AuditDetails
WHERE approvalStatus = 2
ORDER BY auditDate DESC

-- אמור להראות רק ערכים עד 365 ימים
```
