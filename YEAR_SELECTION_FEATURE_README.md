## ?? מימוש טכני

### Backend (C#)

**ReportService.cs:**
```csharp
public async Task<List<KindergartenDetailedReportDto>> 
    GetKindergartensByApprovalStatusAsync(int approvalStatus, int? year = null)
{
    // אם year = null ? שנה אחורה
    // אם year = 2024 ? רק שנת 2024
    
    var dateFilter = year.HasValue 
        ? "AND YEAR(ad.auditDate) = @Year"
        : "AND ad.auditDate >= DATEADD(YEAR, -1, GETDATE())";
}
```

**SQL Queries:**

```sql
-- ללא year (ברירת מחדל)
WHERE ad.approvalStatus = @ApprovalStatus
AND ad.auditDate >= DATEADD(YEAR, -1, GETDATE())

-- עם year=2024
WHERE ad.approvalStatus = @ApprovalStatus  
AND YEAR(ad.auditDate) = 2024
```
